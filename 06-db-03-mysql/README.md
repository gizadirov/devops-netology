экономии места на диске;- размер буффера с незакомиченными транзакциями 1 Мб;- буффер кеширования 30% от ОЗУ;- размер файла логов операций 100 Мб.Приведите в ответе изменённый файл `my.cnf`.## Ответ```commandline[mysqld]skip-host-cacheskip-name-resolvedatadir=/var/lib/mysqlsocket=/var/run/mysqld/mysqld.socksecure-file-priv=/var/lib/mysql-filesuser=mysqlinnodb_flush_method = O_DSYNinnodb_file_per_table = ONinnodb_log_buffer_size = 1Minnodb_buffer_pool_size = 30%innodb_log_file_size = 100Mpid-file=/var/run/mysqld/mysqld.pid[client]socket=/var/run/mysqld/mysqld.sock!includedir /etc/mysql/conf.d/default-character-set=utf8```---
